Developer's supervision

There is a list of websites where PopUpOFF is not working by default. These websites do not show popups but use the same mechanic to display information to users. If you enable it there accidentally, you could hide some important information from yourself.

Disabling supervision is highly not recommended. If you do it, responsibility for further usage would be only on you.
