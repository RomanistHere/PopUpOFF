Delicate mode

Kudos to Davide Repetto for sharing idea of this mode.

This mode reassigns CSS position rule for "fixed" elements. You can pick, what rule will be assigned.

If you're not familiar with CSS, here's a difference: Absolute will "fix" overlays where they first appeared (on top of content). Relative and Static will move sticky elements somewhere else, usually the top of bottom of the page.
